# Block コンポーネント

デザインシステムコンポーネントを組み合わせて作成された、実用的な Block コンポーネントのコレクションです。

## 概要

Block コンポーネントは、Mantine UI の複数のコンポーネントを組み合わせて作られた、より高レベルな UI パターンです。よく使われる UI パターンをコンポーネント化することで、開発効率の向上と一貫性の確保を実現します。

## 特徴

- **組み合わせ**: 複数の Mantine コンポーネントの連携
- **実用性**: 実際のプロジェクトでよく使われるパターン
- **効率性**: 定型的な UI パターンの再利用
- **統一感**: デザインシステムに基づいた一貫したスタイル

## コンポーネント一覧

### 基本レベル

- **LoginForm** - TextInput + PasswordInput + Button + Checkbox
- **SearchBox** - TextInput + ActionIcon + Menu（フィルタ機能）
- **UserCard** - Avatar + Text + Badge + Group（プロフィール表示）
- **NotificationBanner** - Alert + CloseButton + ActionIcon
- **FilterGroup** - MultiSelect + DatePicker + NumberInput + Button

### 中級レベル

- **ProfileEditForm** - 複数の Input + FileInput + Switch + Button 群
- **DataFilterPanel** - Select + DatePicker + Checkbox 群 + Range Slider
- **CommentSection** - Textarea + Button + Timeline + Menu

### 上級レベル

- **DashboardLayout** - Grid + Card + Navbar + Aside + Header
- **MultiStepForm** - Stepper + 各種 Input + Progress + Button 群
- **DataVisualization** - Select + DatePicker + Chart + Table 組み合わせ

## 使用方法

各 Block コンポーネントページでは以下の情報を提供しています：

1. **ライブデモ** - 実際に動作するコンポーネント
2. **構成要素** - 使用されている Mantine コンポーネント
3. **実装コード** - TypeScript + Mantine UI
4. **プロパティ** - カスタマイズ可能な設定
5. **バリエーション** - 異なる用途での使用例
6. **関連パターン** - 類似する Block コンポーネント

## 設計原則

- **モジュラー**: 独立したコンポーネントとして使用可能
- **カスタマイズ可能**: プロジェクトに応じた調整が容易
- **レスポンシブ**: 様々なデバイスサイズに対応
- **アクセシブル**: キーボード操作とスクリーンリーダー対応
