# Custom コンポーネント

デザイントークンやガイドラインを用いて作成された、独自コンポーネントのコレクションです。

## 概要

Custom コンポーネントは、Mantine UI の基本コンポーネントを拡張・カスタマイズして作られた独自のコンポーネントです。デザインシステムの一貫性を保ちながら、特定の用途に特化した機能を提供します。

## 特徴

- **拡張性**: Mantine UI ベースで高いカスタマイズ性
- **一貫性**: デザインシステムに準拠した統一感
- **再利用性**: プロジェクト間での再利用が可能
- **アクセシビリティ**: WCAG 準拠のアクセシブルな実装

## コンポーネント一覧

### 基本レベル

- **AdvancedCombobox** - 検索履歴・お気に入り機能付き Combobox
- **ColorPalettePicker** - カラーパレット選択 UI
- **ProgressSteps** - ステップ表示付き Progress
- **AnimatedButton** - アニメーション強化 Button
- **RichTextInput** - リアルタイムバリデーション付き TextInput

### 中級レベル

- **DataTable** - 高機能データテーブル
- **FileUploadZone** - ドラッグ&ドロップ対応アップロード
- **CodeEditor** - シンタックスハイライト付きエディタ

### 上級レベル

- **KanbanBoard** - ドラッグ&ドロップかんばん
- **ChartWidget** - インタラクティブチャート
- **CalendarScheduler** - 予定管理カレンダー

## 使用方法

各コンポーネントページでは以下の情報を提供しています：

1. **ライブデモ** - 実際に動作するコンポーネント
2. **実装コード** - TypeScript + Mantine UI
3. **プロパティ** - 利用可能な props と設定
4. **カスタマイズ例** - 様々な使用パターン
5. **関連コンポーネント** - 組み合わせ提案
